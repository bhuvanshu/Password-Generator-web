# Password Generator (Web based)

A simple and secure web-based password generator built using **HTML, CSS, and JavaScript**.  
It allows users to generate strong random passwords with a single click.

---

## Features
- Generates 16-character strong passwords
- Uses uppercase, lowercase, numbers, and special characters
- Clean and responsive UI
- Runs fully in the browser (no backend required)

---

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

---

## How to Use
1. Open the application in a browser
2. Click the **Generate Password** button
3. Copy the generated password
